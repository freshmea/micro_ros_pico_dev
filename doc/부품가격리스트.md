# 부품 가격 리스트

---

## 필요 기능

- 전원: LiPo 3.7V (18650 규격 추정)
- MCU: Pico2W, ESP32-CAM
- 센서: TTP223 터치 센서
- 구동: SG90 서보 모터
- 디스플레이: SSD1306 OLED
- 조명: WS2812 Neopixel
- 알림: Passive buzzer + 버튼
- 케이스

## 전원 규격

- pico2w : 1.8V ~ 5.5V ( w)
    > VSYS is the main system input voltage, which can vary in the allowed range 1.8V to 5.5V, and is used by the on-board
    SMPS to generate the 3.3V for the RP2350 and its GPIO
- sg90 servo : 4.8 V (~5V)
- ssd1306 : (doc 내 전원 규격 미기재)
- ttp223 touch sensor : 2.0V ~ 5.5V DC
- ws2812 neopixel : VDD 4.5V ~ 5.5V (WS2812.pdf 기준)
- passive buzzer : 3.3V (BUZZER_README 기준)
- esp32-cam : (doc 내 전원 규격 미기재)
